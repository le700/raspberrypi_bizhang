### 🔑 核心改进

#### 1. 配置管理

使用 `@dataclass` 定义所有配置，支持类型注解：

```python
@dataclass
class SensorConfig:
    ULTRASONIC_MIN: int = 20
    ULTRASONIC_MAX: int = 500
    ULTRASONIC_AVG_FRAMES: int = 6
    
@dataclass
class YOLOConfig:
    ENABLED: bool = True
    VERSION: str = 'v12n'
    CONFIDENCE_THRESHOLD: float = 0.5
```

#### 2. 日志系统

单例模式的 `RobotLogger`：

```python
class RobotLogger:
    _instance = None
    
    @classmethod
    def get_instance(cls, name="TonyPi", level=logging.INFO):
        if cls._instance is None:
            cls._instance = cls(name, level)
        return cls._instance
```

#### 3. 线程安全

`ThreadSafeValue` 和 `ThreadSafeFlag` 封装：

```python
class ThreadSafeValue:
    def __init__(self, value):
        self._value = value
        self._lock = threading.Lock()
    
    def get(self):
        with self._lock:
            return self._value
    
    def set(self, value):
        with self._lock:
            self._value = value
```

#### 4. 状态机设计

```python
class RobotState(Enum):
    INIT = 0
    PATROL = 1
    OBSTACLE_DETECTED = 2
    AVOIDING = 3
    FALL_RECOVERY = 4
    STOPPED = 5
```

class BayesianFusion:
    def __init__(self):
        self.prior = 0.5
        self.likelihood_ultrasonic = 0.9
        self.likelihood_yolo = 0.7
    
    def fuse(self, ultrasonic_present, yolo_present):
        # 贝叶斯公式

class MemoryPool:
    def __init__(self, max_size=10):
        self.pool = []
        self.max_size = max_size
    
    def get(self):
        if self.pool:
            return self.pool.pop()
        return None
    
    def recycle(self, obj):
